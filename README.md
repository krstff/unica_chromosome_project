# Chromosome Segmentation with YOLOv8

Train a YOLOv8n-seg model to segment chromosomes from microscopy images.

## Quick start

1. **Install dependencies** — Cell 0 (`pip install kagglehub torch torchvision ultralytics`)
2. **Create directories** — Cell 1 (runs automatically, creates `data/` and `runs/`)
3. **Set Kaggle credentials** — Cell 2 (set env vars or drop `kaggle.json` here)
4. **Download dataset** — Cell 3 (~990 MB from Kaggle)
5. **Split & convert labels** — Cell 4 (XML → YOLO format, distributes into train/val/test)
6. **Generate config** — Cell 5 (writes `mydataset.yaml`)
7. **Train** — Cell 6 (`yolov8n-seg`, 100 epochs, batch=8)
8. **Validate** — Cell 7
9. **Inference + visualize** — Cell 8

## Project structure

```
├── proj.ipynb              # Main notebook
├── mydataset.yaml          # YOLO dataset config (generated by Cell 5)
├── data/
│   ├── raw/                # Downloaded Kaggle dataset (Cell 3)
│   │   └── 24_chromosomes_object/
│   │       ├── JEPG/               # Raw images (.jpg)
│   │       └── annotations/        # Pascal VOC XMLs
│   ├── train/              # Split by Cell 4
│   │   ├── images/         # .jpg files
│   │   └── labels/         # YOLO .txt files (converted from XML)
│   ├── val/                # same structure
│   └── test/               # same structure
└── runs/                   # Training outputs (weights, plots, logs)
```

## Requirements

- Kaggle account + API credentials (`kaggle.json` or env vars)
- GPU recommended (A100 in original; batch=8 works on most GPUs)
